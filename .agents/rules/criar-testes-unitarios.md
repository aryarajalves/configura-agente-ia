# Regra Teste unitario

Toda vez que você criar uma funcionalidade nova nesse sistema preciso que também crie testes unitarios para poder validar se o sitema vai está funcionando e encontrar possiveis erros.

Toda vez que você atualizar alguma coisa do sistema precisa que tamém crie, ou atualize os testes unitarios para que continuem funcionando.
